The querySelector() -> method returns the first element that matches a specified CSS selector inside the method given as an argument. It searches in the entire document 
The querySelectorAll() -> method returns all the elements

Q.1) Explain the difference between querySelector() & querySelectorAll().
A.1) querySelector() -> returns the first matching values whereas querySelectorAll() will return all return element
