JavaScript event delegation is a simple technique by which you add a single event handler to a parent element in order to avoid having to add event handlers to multiple child elements

Q.1) How event bubbling or capturing is useful in the practical programming world ?
A.2) When you have a deep level elements and you want to handle top level you can use bubbling or capturing
Q.2) Which property can be used to check the element type on which the event is triggered ?
A.3) To check the element type we use event.target.tagName
Q.3) How can you get value of the element which raised the event ?
A.3) Depending on the type of the element we can use the following properties like event.target.innerHTML, event.target.value
