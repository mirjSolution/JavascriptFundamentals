Q.1) What is the scope of a variable?
A.1) Scope is the lifetime or availabilty of a variable 
Q.2) Function Scope Vs Block Scope
A.2) ES5 has a function scope & that is about hoisting, ES6 does not have 'hoisting' it has a block scope, if you declare variable with 'var' keyword hoisting will be there.
Q.3) What is hoisting in javascript ?
A.3) A process which happening behind the scene, internally it is bringing the declarations on top
Q.4) How does block scope work ?
A.4) It is nothing but the brackets {}, an if condition, for loop, do while loop etc. that is a block creates seperate scope for declarations existing in that block.
